CSS (Cascading Style Sheets)

used to format the layout of a page, can control color, font, size, positioning, images and more

can be used inline, interally or externally

Inline

this is when the style attribute is used inside an HTML element; used to apply a unique style to a single element

Internal

this is when a style element is used in the head section; used to define a style for a single HTML page

External

this is when a link element is used in the head section to link to an external CSS file; used to define a style for many HTML pages
